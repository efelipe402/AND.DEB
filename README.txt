git init :
Inicializa  y toma una especie de foto inicial donde se esta trabajando.


git add :
agrega un archivo nuevo o modificado  para depues hacer el commit

 git commit :
actuliza o agrega un nuevo archivo al origin y asi crea un codigo hexadecimal que se puede ser llamado nodo para asi identificar cada uno de ellos y sus etapas

git status :
que archivos han cambiado, cuales estan listos y culaes no ademas de su seguimientos


git diff :
Muestra que index ha sido modificado y que fue agregado.

git log :
muestra los nombre de las referencias de cualquier commit


git checkout :
Actualizaciones de los archivos en el directorio de trabajo para que coincida con la versión en el índice o el árbol especificado.

git push :
actuliza referencias remotas usando las locales .

git remote :
Gestiona los repositorios 

git clone :
clona  un repositorio.
